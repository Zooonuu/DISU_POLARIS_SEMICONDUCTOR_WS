# Roadmap

## Gate 0 — 환경 확인
- Sentaurus/Silvaco 버전 확인
- 3D Process, Device, AC, MixedMode 라이선스 확인
- Python 환경 확인

## Gate 1 — Baseline Process
종료 조건: 대칭 Spacer single-fin 3D 구조 파일 생성 및 구조 치수 검증

## Gate 2 — Baseline Device
종료 조건: NMOS/PMOS의 Id–Vg와 Id–Vd 수렴, 기본 metric 추출

## Gate 3 — Proposed Process
종료 조건: Source-short / Drain-long composite Spacer 구조 생성

## Gate 4 — Proposed Device
종료 조건: Baseline과 동일 bias/mesh 기준 비교 완료

## Gate 5 — Circuit
종료 조건: Inverter VTC, delay, power 및 RO 결과 확보

## Gate 6 — Optimization
종료 조건: DOE, Pareto front, 검증 후보 확보

## Gate 7 — Robustness & R2R
종료 조건: variation 및 open/closed-loop 결과 확보

## Gate 8 — Submission
종료 조건: 보고서, 포스터, 발표자료, 재현성 체크리스트 완료
