# 03 Baseline Process

목표: 대칭 Si3N4 Spacer를 갖는 single-fin 3D FinFET Golden Deck 작성.

권장 순서:
1. substrate와 STI
2. fin patterning
3. gate oxide/high-k
4. gate
5. symmetric spacer
6. source/drain 및 extension
7. contacts
8. mesh
9. structure export

`*.template` 파일은 실행 가능한 완성 Deck이 아니라, 사용 중인 TCAD 버전에 맞춰 채우는 구조 설계서이다.
