# DISU FinFET Reference Pack

파일 구성:

- `REFERENCES.md`: 모든 추적 가능한 출처와 프로젝트 연관성
- `references.bib`: LaTeX/BibTeX 인용 파일
- `SOURCE_MAPPING.csv`: 출처–아이디어–중복 위험 매핑표
- `RESEARCH_ETHICS_AND_NOVELTY.md`: 표절·신규성 위험과 안전한 표현

권장 프로젝트 위치:

```text
DISU_WS/
├── references/
│   ├── REFERENCES.md
│   ├── references.bib
│   ├── SOURCE_MAPPING.csv
│   └── RESEARCH_ETHICS_AND_NOVELTY.md
```

유료 또는 저작권 보호 논문의 PDF 원문은 이 패키지에 포함하지 않았다.
DOI를 통해 학교 도서관 또는 출판사에서 합법적으로 열람한다.
